# Zag.js Documentation

This directory contains the `llms.txt` file for Zag.js documentation.

## Setup

1. Edit `llms.txt` and add your Zag.js component documentation links
2. Follow the llms.txt specification format:
   - Use `##` headings for file groups
   - Format links as `[Name](URL)` with optional descriptions after `:`
   - Keep content under 10,000 tokens per section

## Format Example

```markdown
## Core Components

[Accordion](https://zagjs.com/components/accordion)
: A vertically stacked set of interactive headings that each reveal a section of content.

[Dialog](https://zagjs.com/components/dialog)
: A window overlaid on either the primary window or another dialog window.
```
