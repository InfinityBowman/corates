# CoRATES Architecture Diagrams

Mermaid diagrams to help new contributors understand the CoRATES application architecture.

## How to View

- **VS Code**: Open any `.md` file and use Markdown preview (Cmd+Shift+V)
- **Browser**: Open [viewer.html](viewer.html) for a rendered view of all diagrams
- **GitHub**: Diagrams render automatically in the GitHub UI
